# BMSIT_CHATBOT
The BMSIT Admission Chatbot (v1) is a RAG-based CLI system that answers college admission queries using indexed PDFs. It features a two-layer router for document/pipeline selection, 5 retrieval strategies (FAISS, BM25, Hybrid, CrossEncoder, Page-level), OCR fallback, and auto-evaluation with tone rewriting. v2 will add FastAPI + Web UI.
